
# 流程控制

### 循环语句

> `while([条件表达式]){}`
>
> `do-while([条件表达式])`:至少执行一次
>
> `for(;循环条件;){}`
>
> `for(声明变量 in 对象){}`: 枚举对象属性(对象不可以为null,undefined)
>
> ```js
> for (let item of 'grh') {
>     console.log(item)
> }
> // grh
> ```

### 跳转控制语句

> return
>
> break
>
> continue

### 选择语句

> ```js
> if([条件表达式]){
>  ...
> }else if[条件表达式]){
> ...
> }else{
> ...
> }
> ```
>
> ```js
> switch([条件表达式]){
> case 标签1:
> 代码片段1;
> break;
> ...
> default: 
> 代码片段n;
> }
> ```

### 异常处理语句

> throw: 主动抛出异常
>
> try: 指明需要处理的代码段
>
> catch: 捕获异常
>
> finally: 后期处理
