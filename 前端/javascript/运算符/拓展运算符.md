# 拓展运算符

## 数值运算符

> ES2021

| 符号   | 名称           | 描述                                  | EG                                   |
| ------ | -------------- | ------------------------------------- | ------------------------------------ |
| `_`    | 数据分割运算符 | _可以用来增加数值的易读性             | `100_00 === 10000, 0x11_1 === 0x111` |
| `??`   | 零合并运算符   | 和\|\|类似, 但是不会过滤掉false, 0    | `0 ?? 'default'// 0`                 |
| `?.`   | 可选链操作符   | 在引用null 活undefined不会报错        | `obj.b?.c,`                          |
| `#`    | 私有方法/属性  | 标记私有方法或变量                    |                                      |
| `~~`   | 双位运算符     | 取整                                  | `~~4.5 // 4`                         |
| `void` | void 运算符    | 使用改符号的方法, 返回值只有undefined | `void function fn(){...}`            |
| `**`   | 求幂运算符     | 求幂                                  | `a**b=>a^b`                          |

```js
()=>{
  // _
  console.log( 100_00 === 10000, 0x11_1 === 0x111 , 0b11_1 === 0b111 )
 },
 () =>{
  // ??
  console.log( 
   undefined || 'default',
   null || 'default',
   false || 'default',
   0 || 'default',
   
   undefined ?? 'default',
   null ?? 'default',
   false ?? 'default',
   0 ?? 'default',
   )

   let a = { b: null, c: 10 }
   a.b = a.b ?? 20
   a.c = a.c ?? 20
   console.log(a)
 },
 ()=> {
  // ?.
  const obj = {
   a: 'foo',
   b: {
    c: 'bar'
   }
  }  
  console.log(
   obj.b?.c,
   obj.d?.c,
   obj.func?.(),
  )
 },
 () => {
  // 私有方法/属性 #
  class Person{
   getDesc(){
    return this.#name + " " + this.#getAge()
   }

   #getAge(){
    return this.#age
   }
   get #name(){
    return 'foo'
   }
   #age = 23
  }
  const a = new Person()
  console.log(a.age)
  console.log(a.getDesc())
 },
 ()=>{
  // ~~
  console.log( ~~4.5 , ~~-4.5)
 },
 ()=>{
  // void
  const customMethod = ()=> 123
  const func = () => void customMethod();
  console.log(func())
 },
 ()=>{
  // **
  console.log( 2**3 )   
 },
```
